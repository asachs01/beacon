# Install in 2 Minutes

**Duration:** ~30 seconds

---

## Scene 1: Add Repository (0s–10s)

**Scene:** Home Assistant Settings > Add-ons > Add-on Store. Click the three-dot menu, "Repositories," paste the Beacon repo URL, click Add.

**Voice:** "Here's how fast this is. In Home Assistant, go to the add-on store, add the Beacon repository URL — it's one line, you'll find it in the README."

**Transitions:** Fade in to HA settings, show the repository add flow.

---

## Scene 2: Install Add-on (10s–20s)

**Scene:** Beacon appears in the add-on list. Click it, click "Install." Progress bar fills. Click "Start."

**Voice:** "Now Beacon shows up in your add-ons. Hit install, wait a few seconds, hit start. That's it — it's running."

**Transitions:** Click through install flow, progress bar animation.

---

## Scene 3: Open and Done (20s–30s)

**Scene:** Click "Open Web UI" or find Beacon in the sidebar. The dashboard loads with the family's data already populated from HA.

**Voice:** "Open it up and your calendars, lists, and weather are already there — pulled straight from Home Assistant. Mount a tablet, bookmark it on your phone, and your family dashboard is live. Two minutes, zero config."

**Transitions:** Dashboard loads, hold on the full view, fade to Beacon logo.
