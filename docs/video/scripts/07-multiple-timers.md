# Kitchen Timers

**Duration:** ~35 seconds

---

## Scene 1: Creating a Timer (0s–12s)

**Scene:** Open the timers view. Tap "Add Timer," name it "Pasta," set it to 10 minutes. Start it.

**Voice:** "Cooking dinner and juggling three things at once? Add a timer — give it a name, set the time, and go. Here's one for pasta, ten minutes."

**Transitions:** Fade in to timers view, show the creation flow.

---

## Scene 2: Multiple Timers Running (12s–25s)

**Scene:** Add a second timer — "Garlic Bread, 8 minutes." Now two timers are counting down simultaneously. Both visible on screen with their names and remaining time.

**Voice:** "Now add another one — garlic bread, eight minutes. Both running at the same time, both visible right here. No confusion about which timer is for what."

**Transitions:** Second timer appears alongside the first, both counting down.

---

## Scene 3: Timer Goes Off (25s–35s)

**Scene:** One timer hits zero. An alert or sound indicator shows. Tap to dismiss it. The other timer keeps running.

**Voice:** "When a timer's done, you'll know. Dismiss it, and the others keep right on counting. Simple as that."

**Transitions:** Timer alert animation plays, dismiss, fade out.
