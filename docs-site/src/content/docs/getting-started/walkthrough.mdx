# Visual Walkthrough

This page is a screenshot-guided tour of Beacon. Every major view is shown at both desktop (1280x800) and mobile (390x844) sizes so you know exactly what to expect.

> Screenshots are from a live Beacon instance running as a Home Assistant add-on.

---

## Dashboard

The dashboard is your family's home screen. At a glance you see a large clock, today's date, a personalized greeting, today's events from all connected calendars, a meal plan bar, weather chip, task summary, chores panel, and the leaderboard.

### Desktop

![Dashboard on desktop](/docs/images/dashboard-desktop.png)

The Home Assistant sidebar is on the far left. Beacon's own icon sidebar sits inside the add-on panel with quick access to every view.

### Mobile

![Dashboard on mobile](/docs/images/dashboard-mobile.png)

On mobile, Beacon's navigation moves to a bottom tab bar: Dashboard, Calendar, Chores, Music, and a **More** menu for everything else.

---

## Calendar

A full week view with color-coded events from every connected HA calendar entity. Filter pills toggle calendars on/off. Inline weather shows each day's high/low. Tap any empty time slot to quick-add an event.

### Desktop

![Calendar on desktop](/docs/images/calendar-desktop.png)

### Mobile

![Calendar on mobile](/docs/images/calendar-mobile.png)

On mobile the calendar shows a scrollable 3-day view with page navigation.

---

## Chores

Assign recurring or one-off chores to family members. Each member gets a card showing their avatar and assigned tasks. Tap a chore to mark it done — completions feed into the Leaderboard.

## Desktop vs. Mobile Navigation

| | Desktop | Mobile |
|---|---------|--------|
| **Layout** | Vertical icon sidebar | Bottom tab bar + More |
| **Primary tabs** | All views visible | Dashboard, Calendar, Chores, Music |
| **Overflow** | None needed | More menu for Lists, Tasks, Leaderboard, Photos, Timer, Weather, Settings |
